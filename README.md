# algogrit.com

Contains content for [my site](https://algogrit.com).

Built using [Lovable](https://lovable.dev).
Hosted on [Netlify](https://www.netlify.com/) at
[resume.algogrit.com](https://resume.algogrit.com/).

## Dev

```bash
npm install
npm run dev

npm run build # To build locally
```

## Deployment

Netlify automatically deploys the `master` branch from
[`algogrit/gauravagarwalr.com`](https://github.com/algogrit/gauravagarwalr.com).

- Build command: `npm run build`
- Publish directory: `dist`
- Node.js version: defined in `.node-version`

Push or merge changes into `master` to deploy them to production.
